std::vector< Polynom > generate_irrpolys ( std::size_t const  amount,
BasicInt const  max_defect 
)

Generates vector of first amount least-degree irreducible polynomials over GF(2).

Let \([p_n(x)]_{n=0}^{s-1}\) be a sequence of irreducible polynomials over GF(2). To generate a base 2 (t,s)-sequence using \([p_n(x)]\), we need the sum \(\sum_{k=0}^{s-1}(deg(p_k) - 1) = t\) to be smaller than \(\max\{m\} =\) NBITS.
Let's say, that \(p(x) = \sum_{i=0}^{deg(p)}p_i\cdot x^i,\ \ p_i \in\) GF(2) is equal to the integer \(\overline{p} = \sum_{i=0}^{\log_2 \overline{p}}p_i\cdot 2^i\).
To generate \([p_n(x)]_{n=0}^{s-1}\) we'll test every polynomial, equal to integers \(\overline{p}\) running increasingly from 3 with step 2, with Berlekamp's irreducibility test, supposing that \(p_0(x) = x\).

Returns
Vector of amount least-degree irreducible polynomials.
