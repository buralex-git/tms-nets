Detailed Description

template<typename UIntType, unsigned int NBITS>
class sequences::Niederreiter< UIntType, NBITS >

Provides an interface to generate an elements of a NBITS-bit base 2 (t,s)-sequences in a fixed-dimensional real space.

Template Parameters
UIntTypeNumber of available digits to store the integer \(Q_n = x_n\cdot 2^\textbf{NBITS+1},\ x_n^{(i)} < 1\ \ \forall i\).
NBITSHence the maximum value of m parameter of produced base 2 (t,m,s)-net equals to NBITS as there can be generated only \(2^\textbf{NBITS}\) unique numbers.

Niederreiter ( std::vector< BasicInt > const &  degrees_of_irred)

Constructor, preparing the generation of base 2 (t,s)-sequence poitns with parameters, specified by vector of degrees of irreducible polynomials.

Let \(\{d_i\}_{i=0}^{s-1}\) is equal to degrees_of_irred, where \(s = \)degrees_of_irred.size().
Then \(t = \sum_{i=0}^{s-1} (d_i - 1)\).

void initialize_c ( void  )
private

Initializes lookup tables of c(i,j,r) coefficients.

This program calculates the values of the constants \(c^{(i)}_{jr}\) denoted as c(i, j, r). As far as possible, Niederreiter's notation is used.
For each value of i, we first calculate all the corresponding values of c (all these values are either 0 or 1) and pack them into the 2D-array c_cj thats denoting \(С^{(i)}_r\)
in such a way that c_cj[i][r] holds the values of c(i, j, r) for the indicated i and r and for every j from 0 to (NBITS - 1).
The most significant bit of c_cj[i][r] (not counting the sign bit) is c(i, 0, r) and the least significant bit is c(i, NBITS - 1, r) that is equivalent to
\(С^{(i)}_r = \sum\limits_{j=0}^{\textbf{NBITS}-1} c^{(i)}_{jr}\cdot 2^{\textbf{NBITS}-1-j}\).
